# spring-batch-data

> Base de dados para projeto Spring Batch.

- Executar do docker compose:

```bash
docker-compose up -d
```

1 - Crie um arquivo '.env' com base no '.env.example'

2 - Execute os comandos:

- para levantar o banco

```bash
make up
```

- para 'desligar'

```bash
make down
```

- Ver logs

```bash
make logs
```
